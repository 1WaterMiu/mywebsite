# Jiahuiyu Fang Personal Website

A lightweight static academic homepage inspired by the structure and tone of
[`al-folio`](https://github.com/alshedivat/al-folio).

## Local Preview

Run this from the project directory:

```powershell
python -m http.server 8080
```

Then open:

```text
http://localhost:8080
```

## GitHub Pages

This site can be published directly from the repository root using GitHub Pages.
Use the `main` branch and `/ (root)` as the Pages source.

## Content

- `index.html`: homepage content
- `assets/css/styles.css`: visual styling and responsive layout
- `assets/js/main.js`: mobile navigation and active-section highlighting
- `assets/img/profile-home.png`: homepage portrait
- `assets/img/painting.jpg`: painting preview image
